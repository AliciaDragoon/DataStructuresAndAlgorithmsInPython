# 根据渐近增长速率对下面的函数进行排序。

# 4n(log n) + 2n  2^10  2^(log n)

# 3n+100(log n)   4n    2n

# n²+10n          n^3   n(log n)

1. \(2^{10}\)  
   (常数阶 \(O(1)\))

2. \(2^{\log n}\)  
   (等价于 \(n\)，线性阶 \(O(n)\))

3. \(2n\)  
   (线性阶 \(O(n)\)，系数最小)

4. \(3n + 100 \log n\)  
   (线性阶 \(O(n)\)，系数次小)

5. \(4n\)  
   (线性阶 \(O(n)\)，系数最大)

6. \(n \log n\)  
   (线性对数阶 \(O(n \log n)\)，基础形式)

7. \(4n(\log n) + 2n\)  
   (线性对数阶 \(O(n \log n)\)，系数更大)

8. \(n^2 + 10n\)  
   (平方阶 \(O(n^2)\))

9. \(n^3\)  
   (立方阶 \(O(n^3)\))
